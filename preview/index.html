Production

This is the live production system. V3 is a Model Context Protocol server with 21 tools and zero UI maintenance. It runs natively inside Claude.ai, Claude Desktop, and Claude Code.

How to Use the Live Build

V3 has no web interface — it's a pure data layer that plugs into Claude natively.

  1. Open Claude.ai — Visit claude.ai in your browser (recommended)
  2. Or use Claude Desktop/Code — If the MCP server is configured locally
  3. Ask questions naturally:

    Try these queries:

    • "How's the portfolio looking?"
    • "Show me churn risks for this month"
    • "Who needs renewal follow-up?"
    • "Generate call prep for [customer name]"
    • "What's our NRR this quarter?"

Setup (If Not Configured)

If the MCP server isn't running locally, start it:

cd tendium-mcp
npm install
npm run dev

Then configure Claude Desktop to connect. See the V3 Technical Spec for MCP configuration details.

What Makes V3 "Live"?

  • Zero UI maintenance: Same 21 tools, same 7 data sources, but Claude is the interface
  • Multi-surface: Works in Claude.ai, Claude Desktop, Claude Code, and future Claude surfaces
  • Production-ready: Uses STDIO transport, health scoring, HITL actions
  • The moat: Data stitching logic (not the UI) is the differentiator
