Constraints

  • 6-week sprint ‚Äî V1 by March 6, 2026 (end of Q1)
  • Procurement data moat ‚Äî 90k awards + 14k pipeline, competitors can't access this
  • Company Groups strategy ‚Äî 100k companies from Allabolag, manual in HubSpot today
  • Speed bar ‚Äî Must beat Hugo's 5-hour spreadsheet assembly (target: 30 seconds)
  • Behavioral change required ‚Äî Jens's Sales Graph Explorer set the adoption bar
  • Ground truth principle ‚Äî Raw data access, no abstraction layers (direct APIs)
  • Read-only architecture ‚Äî No writes to source systems, SQLite cache only

Non-Goals

  • Not a CRM (complements HubSpot, doesn't replace it)
  • Not a dashboard (search/explore entities, not view charts)
  • Not a data warehouse (real-time stitching, not pre-aggregation)
  • Not a product feature (internal tooling only)
  • Not user analytics (company-level intelligence, not login tracking)
  • Not AI black boxes (every insight traces to source data)
