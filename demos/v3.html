V3 — MCP Server LIVE

Data layer with zero UI maintenance. Runs inside Claude.ai and Claude Code.

Why V3? V3 proved that the moat isn't the app — it's the data and stitching logic. Same 21 tools, same 7 data sources, but Claude is the interface.

Try It

  1. Open Claude.ai (recommended) or use Claude Desktop/Code
  2. The MCP server should be auto-configured if running
  3. Ask: "How's the portfolio looking?"
  4. Claude will automatically use the 21 tools to answer

Setup (if not already configured)

If the MCP server isn't running, start it:

cd tendium-mcp
npm install
npm run dev

Then configure Claude Desktop to connect to it. See the V3 Technical Spec for MCP configuration details.

What You Can Ask

  • "Show me churn risks for this month"
  • "Who needs renewal follow-up?"
  • "Generate call prep for [customer name]"
  • "What's our NRR this quarter?"
  • "Show usage trends for top 10 customers"

No web UI needed. V3 is a pure data layer that plugs into Claude natively through the Model Context Protocol.
