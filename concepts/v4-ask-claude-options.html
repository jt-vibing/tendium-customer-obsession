V4 "Ask Claude" Options

Interactive demo of three implementation approaches for connecting V4 visual dashboards to Claude's conversational AI. Try each option to see how they work.

Context: V4 focuses on visual dashboards (KPIs, heatmaps, alerts). Instead of building a custom copilot UI, we add contextual "Ask Claude" buttons that open Claude.ai with pre-populated prompts. This gives 95% of the value with 5% of the code.

Option A: Deep Link with URL Parameters Future

Opens Claude.ai with a pre-populated prompt via URL parameter. Smoothest UX, but depends on Claude.ai supporting this feature.

Pros

  • Smoothest user experience
  • Zero clipboard interaction
  • ~5 lines of code per button
  • Fully automated

Cons

  • Requires Claude.ai URL param support
  • Not available today (future feature)
  • No fallback if not supported

Option B: Copy to Clipboard + Open Claude.ai Recommended

Copies contextual prompt to clipboard and opens Claude.ai. User pastes (Cmd+V) and hits Enter. Works today, minimal code.

Pros

  • Works today (no dependencies)
  • ~10 lines of code per button
  • Still contextual and helpful
  • Reliable across browsers

Cons

  • Extra step (paste)
  • Less elegant than Option A
  • Requires clipboard permissions

function askClaude(context) { const prompt = `Using Customer Graph MCP:\n\n${context}`; // Copy to clipboard navigator.clipboard.writeText(prompt); // Show toast showToast('Prompt copied! Opening Claude.ai...'); // Open Claude.ai setTimeout(() => window.open('https://claude.ai/new', '_blank'), 300); }

Option C: Claude Desktop Protocol Handler Experimental

Tries Claude Desktop first (native MCP), falls back to Claude.ai. Best of both worlds if protocol handlers are supported.

Pros

  • Native MCP experience
  • Fallback to web if Desktop not installed
  • Best UX for Desktop users
  • Seamless integration

Cons

  • Complex implementation
  • Depends on protocol support
  • Requires Claude Desktop installed
  • May have browser security warnings

function askClaude(context) { const prompt = `Using Customer Graph MCP:\n\n${context}`; const desktopUrl = `claude://new?mcp=customer-graph&q=${encodeURIComponent(prompt)}`; const webUrl = `https://claude.ai/new`; // Try Desktop first window.location.href = desktopUrl; // Fallback to web after 500ms setTimeout(() => window.open(webUrl, '_blank'), 500); }
